# 🗣️ ForumCraft — Desktop Online Forum

A lightweight, classic web forum built exclusively for desktop environments.  
Crafted with **HTML5**, **Sass**, and powered by **Vite**, ForumCraft focuses on simplicity, speed, and clean structure—without relying on heavy frameworks.

---

## 🖥️ Features

- 💬 **Desktop-First Design**  
  Fixed layout optimized for screens **≥1024px** (non-responsive by design)

- ⚡ **Blazing Fast Development**  
  Powered by Vite for instant hot reload and efficient builds

- 🎨 **Modular Styling with Sass**  
  Organized using variables, mixins, and partials for scalability

- 📁 **Clean Architecture**  
  Maintainable and scalable folder structure

- 🧩 **Classic Forum UI**  
  Includes:
  - Categories
  - Threads
  - Posts
  - User profiles

- 🔒 **Static Frontend**  
  Ready to integrate with any backend or API

---

## 🛠️ Tech Stack

- **Semantic HTML5**
- **Sass (SCSS)**
- **Vite** (bundler & dev server)

---

### 🚀 Getting Started

### 1. Clone the repository

```bash
git clone https://github.com/your-username/forumcraft.git
cd forumcraft
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run development server

```bash
npm run dev
```

### 4. Build for production

```bash
npm run build
```

## 🔌 Backend Integration

ForumCraft is a static frontend, meaning you can connect it to:

* REST APIs
* GraphQL backends
* WebSockets for real-time features

You are free to plug in any backend stack such as:

* Node.js (Express, NestJS)
* Django / Flask
* Laravel
* Firebase / Supabase
